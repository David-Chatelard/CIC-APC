Exercício: Maior
====================


Problema
--------

Faça um programa que recebe uma sequência de 10 caracteres, e mostra qual o caractere com maior valor na tabela ASCII.


Entrada
-------

10 caracteres.


Saída
-----

A saída deve ser o caractere com maior valor na tabela ASCII.


Exemplo(s)
----------

| Entrada               | Saída                 |
|-----------------------|-----------------------|
| abcdefghij | j |

| Entrada                                              | Saída                                                |
|------------------------------------------------------|------------------------------------------------------|
| JIHGFEDCBA | J |

| Entrada                                         | Saída                                         |
|-------------------------------------------------|-----------------------------------------------|
| 1239874560 | 9 |
