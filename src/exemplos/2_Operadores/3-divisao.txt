Algoritmo OperadorDivisão

Início
    Escreva "Divisão:"

    /* Escreva o resultado da operação 3 / 2. A divisão de valores inteiros
       também é um valor inteiro. */
    Escreva "      3 / 2   = ", 3 / 2

    /* Escreva o resultado da operação 2 / 3. */
    Escreva "      2 / 3   = ", 2 / 3

    /* Escreva o resultado da operação 2 / 3.0. A divisão de valores inteiros
       por reais é um valor real. */
    Escreva "      2 / 3.0 = ", 2 / 3.0

    /* Escreva o resultado da operação 2.0 / 3. A divisão de valores reais por
       inteiros é um valor real. */
    Escreva "    2.0 / 3   = ", 2.0 / 3

    /* Escreva o resultado da operação 2.0 / 3.0. A divisão de valores reais
       também é um valor real. */
    Escreva "    2.0 / 3.0 = ", 2.0 / 3.0

    /*  Divisão por zero (0) não é uma operação válida! */
    /* Escreva "    2.0 / 0.0 = ", 2.0 / 0.0 */
